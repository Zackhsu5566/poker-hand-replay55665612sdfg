# Replow

A mobile-first poker hand replay tool. Input your live poker hands and generate shareable video replays.

## Features

- Manual hand history input optimized for mobile
- Visual poker table with animated replays
- Video export for sharing on social media
- Support for HU, 6-max, and 9-max tables

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
